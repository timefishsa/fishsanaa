# موقع وقت السمك

## نظرة عامة

موقع إلكتروني متكامل لبيع الأسماك الطازجة مع نظام طلب عبر واتساب. الموقع يتميز بتصميم عصري وسهل الاستخدام مع دعم كامل للغة العربية.

## المميزات

### الوظائف الأساسية
- عرض قائمة شاملة بالمنتجات المتوفرة (29 نوع من الأسماك والمأكولات البحرية)
- نظام طلب متكامل مع حساب تلقائي للفاتورة
- خيارات متعددة لطرق التحضير (نيئ، صانونة، بروست، موفى، زيت)
- دعم أكواد الخصم
- إرسال الطلبات مباشرة عبر واتساب
- صفحات مقالات تثقيفية عن الأسماك

### التصميم
- تصميم متجاوب يعمل على جميع الأجهزة
- واجهة مستخدم عصرية مع تدرجات لونية
- تأثيرات حركية سلسة
- ألوان متناسقة (أزرق داكن #003080)
- خط Cairo للنصوص العربية

## البنية التقنية

### الملفات الرئيسية
```
waqt-alsamak/
├── index.html              # الصفحة الرئيسية
├── order.html              # صفحة إتمام الطلب
├── article-*.html          # صفحات المقالات (4 صفحات)
├── css/
│   └── style.css           # ملف التنسيقات الرئيسي
├── js/
│   ├── main.js             # سكريبت الصفحة الرئيسية
│   ├── products.js         # بيانات المنتجات وأكواد الخصم
│   └── order.js            # سكريبت صفحة الطلب
└── images/
    ├── logo.png            # شعار الشركة
    └── fish-icon.svg       # أيقونة السمك
```

### التقنيات المستخدمة
- HTML5
- CSS3 (مع Flexbox و Grid)
- JavaScript (Vanilla JS)
- Google Fonts (Cairo)

## كيفية الاستخدام

### التشغيل المحلي
يمكن تشغيل الموقع على أي خادم ويب محلي:

```bash
# باستخدام Python
python3 -m http.server 8080

# أو باستخدام Node.js
npx http-server -p 8080
```

ثم افتح المتصفح على: `http://localhost:8080`

### النشر
الموقع عبارة عن ملفات ثابتة (Static Files) يمكن رفعها على أي خدمة استضافة:
- GitHub Pages
- Netlify
- Vercel
- أي استضافة ويب تقليدية
